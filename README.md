# Markup Test

A responsive profile posts page based on a Figma design. The project is built with semantic HTML, BEM-style CSS, and modular JavaScript without a build step.

## Features

- Responsive header and decorative SVG elements
- Rows and tiles post views
- Animated view switching and interactive card states
- Local Flatpickr date range picker
- Responsive layouts for desktop, tablet, and mobile

## Project Structure

```text
css/                Styles and normalization
fonts/              Local font files
img/                Images, icons, and decorative SVGs
js/modules/         JavaScript feature modules
vendor/flatpickr/   Local Flatpickr library files
index.html          Main page
```
